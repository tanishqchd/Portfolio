Fabrication of Piston Head

I did this project for credits towards my course on Manufacturing Technologies. What this project entailed was that I first measured a piston head from a compressor of a refrigeration machine in our lab, then I modelled it on PTC Creo, created a 2D drawing, and finally fabricated it. The following process was used to fabricate the piston head:

The steps taken to fabricate were: 1. Casting using oversized pattern for allowances.
2. Turning and facing on a conventional lathe machine.
3. Cutting off extra portion.
4. Boring using grooving tool with a 25mm diameter.
5. Rubbing with sandpaper to get rid of coarse material particles on the surface.
6. Marking, punching and drilling with a 20 mm drill bit.
7. Milling to create the U-shaped cuy.
8. Upper groove using 20 mm drill bit.
